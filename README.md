Nombre del Proyecto: Sistema de Gestión de Consultas Médicas

Estudiantes: Celimar Rojas y Nicole Sereno

Descripción del Proyecto:
Sistema de gestión integral para consultorios médicos desarrollado en Java con Programación Orientada a Objetos. Permite gestionar pacientes, médicos, citas, facturación y reportes con una interfaz gráfica sencilla e intuitiva.

Objetivo: Automatizar el flujo completo de un consultorio médico, desde el registro del paciente hasta la entrega de comprobantes, optimizando tiempo y reduciendo errores manuales.

Requisitos del Sistema:

Software Necesario
- Java JDK 8 o superior (JDK 11+ recomendado)

- Sistema operativo: Windows, Linux o macOS

- IDE Utilizado: NetBeans

Paso 1: Instalación y Ejecución:

 - Opción #1:  Clonar el Repositorio en NetBeans
   
1. Abrir NetBeans

2. Ir al menú: Team → Git → Clone...

3. En la ventana emergente, ingresar:

Repository URL: https://github.com/CELIM709/hospital-prog3.git

User: [Dejar en blanco o usar credenciales públicas]

Password: [No necesario para repositorio público]

4. Click en "Next"

5. Seleccionar rama "main" (si está disponible)

6. Elegir carpeta destino donde se descargará el proyecto

7. Click en "Finish"

NetBeans descargará automáticamente todo el proyecto y lo abrirá.
